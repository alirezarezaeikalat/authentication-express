Development

We use MongoDB and node version 14.17.6:
nvm install 14.17.6
nvm use 14.17.6


We use redis for session management. Install redis and then run:
redis-server

The first time, you will need to run:
npm install

run:
cp .env.example .env.

run (using nodemon):
npm run dev

Or

run:
npm run start
